# Distributed Cloud Download Manager

A cloud-native distributed download manager built using:

- FastAPI
- React
- AWS EC2
- Amazon S3
- Amazon RDS (PostgreSQL)
- API Gateway
- CloudFront
- Docker

## Features

- Distributed chunk downloading
- Parallel workers
- Chunk scheduling
- Fault tolerance
- Download progress tracking
- Automatic worker registration
- Cloud storage
- REST APIs
